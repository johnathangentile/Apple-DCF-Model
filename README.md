# Apple-DCF-Model
Built a 5-year Discounted Cash Flow (DCF) valuation model for Apple Inc. in Microsoft Excel using historical financial statement data from Apple's SEC filings.

## Model Components
- Historical Financials
- Assumptions
- Income Statement Forecast
- Free Cash Flow Forecast
- DCF Valuation
- Sensitivity Analysis

## Key Findings
- Intrinsic Value per Share: $124.40
- Market Price at Time of Analysis: ~$295

Based on the assumptions used in this model, Apple's market price exceeded its estimated intrinsic value, suggesting the stock was trading at a premium relative to the model's projected cash flows.

## Notes
The sensitivity analysis evaluates how changes in WACC and terminal growth rate assumptions affect the estimated intrinsic value.
